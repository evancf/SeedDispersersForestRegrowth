To reproduce analyses, include this folder structure within the data folder to house various downloaded files.

-spatial data
--raster
--raster_intermediary
--raster_outputs
--vector

-land_use_dependent_dispersal_data
--fragmentation
--Movebank raw
--tidy

Other data will appear in this folder (that is, "./data/"), including data to be downloaded by the user. Data that should be downloaded and put in this folder:

GPFC_database.xlsx from https://zenodo.org/records/6555216
movebank_metadata.csv available for registered Movebank users. https://www.movebank.org/cms/movebank-main
