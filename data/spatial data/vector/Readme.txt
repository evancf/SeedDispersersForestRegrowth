The script 01_spatial_data_from_gee.R gives details on downloading data to put in this folder. The files that should be downloaded here so that it appears here is the unzipped folder from the Ecoregions dataset available from this link: https://storage.googleapis.com/teow2016/Ecoregions2017.zip. Also see the "About" tab here: https://ecoregions.appspot.com/
